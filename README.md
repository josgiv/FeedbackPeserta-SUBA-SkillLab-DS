## Cara Melihat Feedback

1. Buka folder `Feedback Peserta`
2. Temukan file: `Feedback_NamaPeserta.md`  
   (contoh: `Feedback_Yuliska_Nur_Hanif.md`)
3. Baca setiap bagian feedback yang tersedia, meliputi:
   - Exploratory Data Analysis (EDA)
   - Preprocessing
   - Modeling
   - Evaluation  
     serta rekomendasi perbaikan yang diberikan.
